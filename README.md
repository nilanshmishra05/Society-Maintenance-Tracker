# 🏠 Society Maintenance Tracker

A full-stack MERN (MongoDB, Express.js, React.js, Node.js) web application developed to simplify complaint management in residential societies. Residents can register, log in, submit complaints with images, and track their complaint status. Administrators can manage complaints, update status and priority, and monitor complaint analytics through a dashboard.

---

## 🚀 Features

### 👤 Resident
- Register Account
- Login
- Dashboard
- Create Complaint
- Upload Complaint Image
- View My Complaints
- View Complaint Details
- Profile Page

### 👨‍💼 Admin
- Secure Login
- Dashboard with Analytics
- View All Complaints
- Search Complaints
- Filter Complaints
- Update Complaint Status
- Update Complaint Priority
- View Complaint History
